#### 1. Les controllers d'API


- Faire les controllers d'API pour les entités suivantes (ordre à respecter) :
  - Country :
    - list / index (tout)
    - show (tout)
  - Category  :
    - list / index (tout)
    - show (tout)
  - Game :
    - list / index :
      - game.slug
      - game.name
      - game.thumbnailCover
      - game.price
    - show (tout) :
      - game.*
      - game.countries :
        - country.slug
        - country.urlFlag
        - country.name
      - game.categories :
        - category.slug
        - category.name
      - game.platforms :
        - platform.slug
        - platform.name
      - game.publisher :
        - publisher.name
        - publisher.slug
      - game.reviews :
        - review.createdAt
        - review.content
        - review.title
        - review.rating
        - review.upVote
        - review.downVote
        - review.user :
          - user.name
          - user.profileImage
  - UserOwnGame
  - Review
  - Publisher
  - User
